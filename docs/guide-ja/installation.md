インストール
============

## Composer パッケージを取得する

このエクステンションをインストールするために推奨される方法は [composer](http://getcomposer.org/download/) を使う方法です。

下記のコマンドを実行して下さい。

```
composer require --prefer-dist yiisoft/yii2-imagine
```

または、あなたの `composer.json` ファイルの `require` セクションに

```
"yiisoft/yii2-imagine": "~2.2.0"
```

を追加して下さい。
